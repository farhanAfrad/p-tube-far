Discuss the scope of var, let, and const
Scope in programming means where the variable is accessible.
  • variable declared using var are globally or function scoped.
  • variable declared using let are block scoped. Block scoped means variables are only accessible within the {} braces.
  • variable declared using const are similar in terms of scope to variable declared using let.

What do you mean by REST API?

REST API is a programming interface which gives the fascility to two computer for sharing data. REST API has some methods(GET,POST,PUT/PATCH,DELETE).
By using these methods client side sent request to the server then server contact with the database based on that sent request. For example: if GET request is sent to the server then server contact with the database and send the information about an API resource.
